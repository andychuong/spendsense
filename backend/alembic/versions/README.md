# Alembic Migrations

This directory contains Alembic database migration scripts.

## Usage

To create a new migration:
```bash
alembic revision --autogenerate -m "description"
```

To apply migrations:
```bash
alembic upgrade head
```

To rollback one migration:
```bash
alembic downgrade -1
```

## Migration Files

Migration files are automatically generated by Alembic and should not be manually edited after creation, unless you know what you're doing.
